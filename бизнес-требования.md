## Цель продукта

Создать удобную персональную ленту Telegram-постов, которую пользователь может быстро листать, фильтровать и сохранять без необходимости заходить в каждый канал отдельно.

## Целевая аудитория

Пользователи Telegram, подписанные на большое количество каналов и испытывающие проблему информационного шума.

## Основная ценность

Собрать посты из выбранных Telegram-источников в один чистый интерфейс с быстрым просмотром, сохранением и фильтрацией.

## MVP-функциональность
- Запуск приложения внутри Telegram как Mini App.
- Авторизация пользователя через Telegram.
- Отображение ленты постов из подключенных источников.
- Вертикальное листание карточек.
- Открытие оригинального поста в Telegram.
- Сохранение поста.
- Скрытие поста или источника.
- Настройка списка источников.
- Базовая админка для управления подключенными каналами.
- Backend-сервис для приема, нормализации и хранения постов.
